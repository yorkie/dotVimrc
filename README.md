
dotVimrc
=====================
My personal vi configs

### Installation
```sh
git clone git@github.com:yorkie/dotVimrc.git
```
or
```sh
git clone https://github.com/yorkie/dotVimrc.git
```

### Usage
```sh
make install
```

If you wanna submit your .vimrc config to me, just run `make update` and make a PR here.
